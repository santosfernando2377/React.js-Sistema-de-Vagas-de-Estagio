# Sistema de Cadastro de Vagas de Aprendizagem

## Foco desse projeto é criar um sistema completo para visualização de novas vagas de estágios.

### Estrutura do projeto

* src/assests ( Os assests são imagens, logos e ícones );
* src/pages ( São às páginas do projeto onde reúne componentes, assests e rotas de navegação );
* src/componentes ( São os componentes reutilizados em cada página );
* public/iframe ( São as renderizações de sites externos dentro da minha página );
* server.js ( Utilizado para rodar o projeto após build );

### Bibliotecas utilizadas
* React;
* React-Router-Dom; 
* Styled-Components;
* Express;
